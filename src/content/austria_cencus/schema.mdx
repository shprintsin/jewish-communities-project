## מבנה הנתונים 

| שדה                     | טיפוס       | תיאור                                                                 |
|--------------------------|-------------|------------------------------------------------------------------------|
| `line`                  | מספר שלם     | מספר שורה (מספור פנימי)                                              |
| `locality`              | מחרוזת       | שם היישוב                                                              |
| `district`              | מחרוזת       | שם הנפה (`Bezirkshauptmannschaft`)                                   |
| `juridiction_dicstrict` | מחרוזת       | שם איזור השיפוט (`Gerichtsbezirk`)                                    |
| `type`                  | מחרוזת       | סוג יישוב (`Ortsgemeinden`/`Gutsgebiete`)                             |
| `page`                  | מספר שלם     | מספר העמוד בספר                                                       |
| `serial_no`             | מספר שלם     | מספר סידורי פנימי של היישוב                                           |
| `area_hectares`         | מספר שלם     | שטח היישוב בהקטאר                                                      |
| `population_male`       | מספר שלם     | מספר הגברים                                                           |
| `population_female`     | מספר שלם     | מספר הנשים                                                            |
| `population_total`      | מספר שלם     | סך האוכלוסייה (נתון מקור)                             |
| `religion_catholic`     | מספר שלם     | מספר בני דת קתולית רומית                                              |
| `religion_greek`        | מספר שלם     | מספר בני הדת היוונית-קתולית                                           |
| `religion_jewish`       | מספר שלם     | מספר יהודים                                                            |
| `religion_other`        | מספר שלם     | דתות אחרות                                                             |
| `c1`, `c2`              | לא ידוע      | מספרים שזלגו מעמודות קודמות              |
| `village`               | מספר בינארי  | 1 = יישוב מסווג ככפר                                                  |
| `belong`                | מספר  | האם שייך לאזור גדול יותר (למשל אחוזה)                               |
| `sum_type`              | משתנה קטגורי עבור שורות סכום, המשתנה מזהה את סוג הסכימה|
| `vide`                  | מחרוזת       | הפניה לשורת יישוב אחרת (למשל: `vide 6`)                               |
| `wrong_sum`             | מספר בינארי  | סימון אם סכום הדתות ≠ סכום המגדרים                                   |

---

> ℹ️ שדות `c1` ו־`c2` הן עמודות שבהן יש זליגה של נתונים וצריך לסדר את הנתונים לפי המיקום הנכון שלהם
>  ניתן להשתמש בשדה `wrong_sum` כדי לזהות שורות שבהן יש בעיה בסיכום האוכלוסייה.  
>  השדה `vide` מציין הפניות מתוך הטקסט לשורות אחרות
