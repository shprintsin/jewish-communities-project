# מבנה הנתונים

תיעוד זה מתאר את מבנה הקובץ `final_galiciaV1.csv` – קובץ המרכז את קהילות יהודי מזרח אירופה בגליציה ובפרוסיה.


| שדה              | תיאור                                                                 |
|------------------|------------------------------------------------------------------------|
| `code`           | מזהה טכני ייחודי לכל יישוב (ראו הסבר בהמשך)                         |
| `index`          | קוד היררכי בן 5–6 תווים לפי פרובינציה, מחוז ושם היישוב                |
| `name`           | שם היישוב בפורמט מקוצר                                                |
| `page`           | עמוד באנציקלופדיה שבו מופיע היישוב                                    |
| `year_code`      | קידוד של תיאור זמן ההתיישבות או הקמת הקהילה                          |
| `year_foundation`| שנת ההקמה (אם ידועה במדויק)                                           |
| `prov`           | שם הפרובינציה (למשל: `Galicia`, `Prussia`)                            |
| `dist`           | שם המחוז                                                               |
| `prov_code`      | קוד מספרי של הפרובינציה (למשל: `62` = Galicia)                        |
| `dist_code`      | קוד מספרי של המחוז                                                    |
| `enc_name`       | שם היישוב כפי שמופיע באנציקלופדיה                                     |
| `source`         | ציטוט תיאורי מתוך האנציקלופדיה לגבי ראשית ההתיישבות                   |
| `lon`, `lat`     | קואורדינטות גאוגרפיות של היישוב                                       |
| `jg_id`          | מזהה היישוב במערכת JewishGen                                          |
| `pop_1900`       | נתון אוכלוסייה יהודית סביב שנת 1900                                   |
| `pop_year`       | השנה המתאימה לנתון `pop_1900`                                        |

---

###  `code`: מזהה טכני ליישוב

הקוד נבנה לפי תבנית אחידה:

```
{PROV_abbr}_{DIST_abbr}_{NAME_abbr}
```
כאשר, `PROV_abbr` מציין את שלושת האותיות הראשונות של שם הפרובינציה (כולם ללא תנועות למעט האות הראשונה והאחרונה), `DIST_abbr` מציין את שלושת האותיות הראשונות של שם המחוז, ו־`NAME_abbr` מציין את שלושת האותיות הראשונות של שם היישוב.
במקרים של כפילויות נוספה סיומת מספרית (`_2`, `_3` וכו')

**דוגמות:**

- `PRU_PSN_BDZY` = Prussia / Posen / Bodzanów  
- `GLC_TRK_NYZH` = Galicia / Turka / Wysocko Nizne  

—

### `index`: קידוד היררכי קצר

`index` מורכב מ־5–6 תווים לפי התבנית:
```
{PROV_code}{DIST_code}{NAME_code}
```
לכל פרובינציה יש קוד מספרי יחודי בין 1-99 ולכל ישוב בתוך הפרובינציה יש קוד ייחודי בין 1-99. המספרים בפורמט של שני ספרות.

למשל, עבור הישוב Czchow שנמצא במחוז Brzesko בגליציה:
- פרובינציה: `62` (Galicia)
- מחוז: `63` (Brzesko)
- שם הישוב: `Czc`
- הקוד המתקבל הוא: `6263Czc`

**קידודים לדוגמה:**

| פרובינציה | קוד |
|-----------|-----|
| Prussia   | `61`|
| Galicia   | `62`|
| Bukovina  | `63`|

### `year_code`: תיעוד כרונולוגי

השדה `year_code` מתעד את השנה או התקופה הראשונה של ההתיישבות היהודית או הקמת קהילה, תוך שמירה על מידע לגבי רמת הוודאות.

#### מבנה הקידוד:

```
{chronolical_codde}-{year_code}
```

- `J` = מידע על יהודים פרטיים  
- `C` = מידע על קהילות  
- `F` = תאריך הקמה  
- `M` = אזכור ראשון  
- `E` = קיום רציף  
- `POP` = נתון אוכלוסייה בלבד  

**תבניות זמן:**

| תיאור              | סיומת |
|--------------------|--------|
| שנה מדויקת         | `-YYYY`|
| מאה כללית          | `XX`   |
| תחילת מאה          | `EX`   |
| אמצע מאה           | `MX`   |
| סוף מאה            | `LX`   |

---

#### דוגמאות:

| תיאור                                                      | קוד       |
|-------------------------------------------------------------|-----------|
| Jews arrived in the 13th cent                               | `JF-12XX` |
| Jews first arrived in 1629                                  | `JF-1629` |
| Jews are first mentioned in 1300                            | `JM-1300` |
| A Jewish community was founded in 1860                      | `CF-1860` |
| A Jewish community existed from the 18th cent               | `CE-18XX` |
| Jewish population was 36 in 1878                            | `POP-1878` |

---

> ℹ️ ניתן להשתמש ב־`year_code` לניתוחים בזמן לא מדויק, תוך הבחנה בין קיום פיזי, אזכור טקסטואלי וייסוד רשמי של קהילה.
